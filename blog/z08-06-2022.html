Basic WebGL

WebGL is a graphics library that can be used to render high quality images and graphics on websites. From 2D to 3D images and objects WebGL is a powerful library that allows developers to further the abilities of their websites. To begin with, WebGL requires a canvas to draw graphics on. This canvas needs to be initialized like a normal canvas and can be processed through its context. Then a webGL library needs to be downloaded. This library will have the necessary functions and data structures to allow for higher-level graphics processing, such as matrices, and shaders. WebGL renders graphics using what it calls Shader Programs. A shader program is made of a Vertex Shader, and a Fragment Shader. The Vertex Shader defines the coordinates and position of the vertices of your object. This determines whether the output is a square, rectangle, star or something else. The Fragment shader is used to determine the color or texture of each texel/(pixel of texture) on the WebGL object. These two components of a shader program are then compiled and then combined into a single shader program. Finally, an object/struct can be created that includes the shader program and other attributes, such as its color, and size. This object is what the rest of the program will use to handle the webGL object.
