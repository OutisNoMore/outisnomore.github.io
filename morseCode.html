Morse Chat Bot

Welcome to a fun and engaging way to easily learn morse code!
If this is your first time using morse code, visit this page for basic info: Basics!
Or if you are already familiar with the basics, skip straight to Chatting!

Use the space bar to control the paddle, and backspace to delete one character or space.
The send button should send what you wrote to a chatbot api when I find/make a good one.
